# Research Constitution

## Purpose
This document is the highest authority in the repository. Every architectural, implementation, and experimental decision must comply with it.

## Goal
Develop a lightweight, interpretable speaker attribution system for literary novels using explicit contextual reasoning rather than large language models.

## Core Research Question
How much contextual representation is necessary for accurate literary speaker attribution?

## Principles
- Simplicity before complexity.
- Evidence before intuition.
- Experiments before assumptions.
- Interpretability over opacity.
- Incremental development.
- Reproducibility.

## Scope
Included:
- Speaker attribution
- Candidate ranking
- Lightweight ML
- Discourse/context representation
- Controlled experiments

Excluded:
- Chatbots
- Story summarization
- General narrative understanding
- Emotion recognition
- Building an LLM

## Mandatory Rule
Every new component must answer:
1. Why is it needed?
2. Which experiment motivated it?
3. How will improvement be measured?
4. Is there a simpler alternative?
